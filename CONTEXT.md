# 도메인 용어 사전 (Domain Glossary)

| 용어 (Term) | 영문명 (English) | 설명 (Description) |
| --- | --- | --- |
| **매장 (합주실)** | Studio | 하나의 비즈니스 위치/사업장을 의미합니다. 여러 개의 방(Room)을 포함합니다. (예: "낙원 합주실") |
| **방 (룸)** | Room | 매장 내에 위치하여 유저가 실제로 합주를 진행하는 개별 공간입니다. (예: "A룸(대형)") |
| **장비** | Equipment | 특정 방(Room)에 비치되어 있는 악기 및 음향 기기입니다. |
| **장비 카테고리** | Equipment Category | 장비의 종류를 나타내는 분류 값입니다. (예: 드럼, 기타 앰프, 베이스 앰프 등) 어드민이 동적으로 관리합니다. |
| **제보** | Submission | 유저가 새로운 합주실(Studio)을 시스템에 등록 요청하거나, 기존 합주실에 새로운 방(Room)이나 장비(Equipment)를 추가해 달라고 요청하는 행위입니다. |
| **대기 상태** | Pending | 제보된 데이터가 아직 관리자(Admin)의 검토를 받지 않은 상태입니다. 지도에 노출되지 않습니다. |
| **승인 상태** | Active | 관리자가 제보를 수락하여 일반 지도나 상세 정보에 정상적으로 노출되는 상태입니다. |
| **반려 상태** | Deny | 관리자가 정보의 부정확성 등을 이유로 제보를 거절한 상태입니다. DB에 영구 기록되며 유저에게 사유와 함께 제공됩니다. |
| **반려 사유** | Deny Reason | 관리자가 제보를 반려할 때 남기는 구체적인 거절 사유입니다. |
| **사전 서명된 URL** | Presigned URL | 유저가 서버를 거치지 않고 S3(Supabase Storage)에 직접 안전하게 이미지를 업로드하기 위해 발급받는 임시 인증 URL입니다. |
