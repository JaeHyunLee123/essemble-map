# 도메인 용어 사전 (Domain Glossary)

| 용어 (Term) | 영문명 (English) | 설명 (Description) |
| --- | --- | --- |
| **매장 (합주실)** | Studio | 하나의 비즈니스 위치/사업장을 의미합니다. 여러 개의 방(Room)을 포함합니다. (예: "낙원 합주실") |
| **방 (룸)** | Room | 매장 내에 위치하여 유저가 실제로 합주를 진행하는 개별 공간입니다. (예: "A룸(대형)") |
| **장비** | Equipment | 특정 방(Room)에 비치되어 있는 악기 및 음향 기기입니다. |
| **장비 카테고리** | Equipment Category | 장비의 종류를 나타내는 분류 값입니다. (예: 드럼, 기타 앰프, 베이스 앰프 등) 어드민이 동적으로 관리합니다. |
| **제보** | Submission | 유저가 새로운 합주실(Studio)을 시스템에 등록 요청하거나, 기존 합주실에 새로운 방(Room)이나 장비(Equipment)를 추가해 달라고 요청하는 행위입니다. |
| **대기 상태** | Pending | 제보된 데이터가 아직 관리자(Admin)의 검토를 받지 않은 상태입니다. 지도에 노출되지 않습니다. |
| **승인 상태** | Active | 관리자가 제보를 수락하여 일반 지도나 상세 정보에 정상적으로 노출되는 상태입니다. |
| **반려 상태** | Deny | 관리자가 정보의 부정확성 등을 이유로 제보를 거절한 상태입니다. DB에 영구 기록되며 유저에게 사유와 함께 제공됩니다. |
| **반려 사유** | Deny Reason | 관리자가 제보를 반려할 때 남기는 구체적인 거절 사유입니다. |
| **수정 요청** | Studio Update Request | 유저가 기존에 승인된 합주실(Studio)의 정보(이름, 설명, 지도 링크) 변경을 관리자에게 요청하는 행위입니다. |
| **수정 승인 상태** | Approved | 관리자가 수정 요청을 수락하여 원본 합주실의 정보가 업데이트된 상태입니다. |
| **수정 반려 상태** | Rejected | 관리자가 정보의 부정확성 등을 이유로 수정 요청을 거절한 상태입니다. |
| **사전 서명된 URL** | Presigned URL | 유저가 서버를 거치지 않고 S3(Supabase Storage)에 직접 안전하게 이미지를 업로드하기 위해 발급받는 임시 인증 URL입니다. |

## 도메인 규칙 및 관계 (Relationships)

- **제보 종속성**: 방(Room)과 장비(Equipment)는 부모인 합주실(Studio)이 **승인 상태(Active)**일 때만 추가 제보가 가능합니다. (대기 상태의 합주실에는 하위 제보 불가)
- **반려된 제보의 재제보(Re-submission)**: 제보가 반려(Deny)된 경우, 해당 레코드는 수정되지 않고 히스토리 보존을 위해 영구히 읽기 전용(Read-only)으로 남습니다. 유저가 재제보를 원할 경우 완전히 새로운 레코드를 생성(INSERT)하여 대기(Pending) 상태로 등록합니다.
- **합주실 수정 요청(Studio Update Request)**: 일반 유저(로그인 필요)는 이미 승인된(Active) 기존 합주실의 정보 수정을 요청할 수 있습니다. 방(Room) 및 장비(Equipment)에 대한 수정 제보는 현재 지원되지 않습니다. 관리자(Admin)는 수정 요청을 그대로 승인하거나 일부 수정하여 승인(`Approved`)할 수 있으며, 불합리한 요청은 반려(`Rejected`)할 수 있습니다. 승인 시 원본 합주실 레코드가 업데이트됩니다.
- **북마크 제약**: 사용자는 **승인 상태(Active)**인 합주실만 북마크할 수 있습니다. 대기 중(Pending)이거나 반려된(Deny) 합주실은 북마크가 불가능합니다.
- **동일 좌표 합주실 처리(Overlapping Markers)**: 한 건물 내에 여러 합주실이 존재하여 위경도 좌표가 동일(또는 매우 유사)할 경우, 지도 UI에서는 이를 하나의 마커(또는 클러스터)로 표시하고, 클릭 시 해당 위치에 속한 "합주실 목록"을 선택할 수 있는 UI(리스트 모달/바텀 시트 등)를 제공하여 겹침 문제를 해결합니다.
