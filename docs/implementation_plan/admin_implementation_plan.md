# 어드민 페이지 구현 계획 (Admin)

이 문서는 애플리케이션의 운영 및 합주실 데이터 관리를 위한 관리자 전용 대시보드 구현 계획을 다룹니다.

## 1. 기능 요구사항 (Requirements)
*   **유저 통계 확인**
    *   현재 가입된 전체 유저 수를 대시보드 상단에 표시합니다.
*   **합주실 제보 관리 및 검증**
    *   일반 사용자가 제보한 합주실 중 수락 대기 중(`pending`)인 목록을 확인합니다.
    *   제보 검토 모달 내에 파싱된 위경도(`lat`, `lng`) 좌표를 마커로 표시하는 **미니 네이버 지도**를 노출하여 시각적으로 위치 정확성을 검증합니다.
*   **수락 및 거절 처리**
    *   대기 중인 합주실 정보를 검토한 후, 상태를 수락(`active`) 또는 거절(`deny`)로 변경합니다.
    *   거절 시에는 자주 쓰는 반려 사유 템플릿(상용구)을 선택할 수 있고 필요 시 상세 텍스트를 덧붙여 입력할 수 있는 거절 모달을 제공합니다.
*   **승인 완료된 합주실 삭제**
    *   이미 승인된(`active`) 전체 합주실 목록을 확인하고, 오동작이나 데이터 삭제를 처리할 수 있도록 즉각 상태를 `deny`(또는 soft delete)로 변경하여 일반 유저 지도 노출에서 신속하게 내릴 수 있는 삭제 액션을 제공합니다.

## 2. 데이터베이스 및 API (Backend)
*   **API 엔드포인트 및 변경 사항 (`api_specification.md` 참고)**
    *   `GET /api/admin/stats` - 총 유저 수 카운트 쿼리를 수행합니다.
    *   `GET /api/admin/studios/pending` - `status = 'pending'` 인 스튜디오 목록을 쿼리합니다.
    *   `PATCH /api/admin/studios/:id/status` - 특정 스튜디오의 상태를 `active` 또는 `deny`로 변경합니다. (이미 `active` 상태인 스튜디오의 비활성화/삭제 용도로도 동일하게 사용됩니다.)
*   **보안 및 어드민 생성 제어**
    *   Next.js Middleware를 활용하여 접속한 사용자의 토큰 내 `role`이 `admin`인지 엄격히 검사합니다.
    *   권한이 없는 유저가 `/admin/*` 라우트나 `/api/admin/*` API에 접근할 경우 `403 Forbidden` 처리하고 홈으로 리다이렉트합니다.
    *   회원가입 API(`POST /api/auth/register`)는 악의적인 권한 탈취를 막기 위해 클라이언트 파라미터와 상관없이 사용자의 `role`을 항상 `user`로만 강제 고정하여 저장합니다.
    *   최초 어드민 계정의 생성 및 권한 부여는 Supabase 대시보드 또는 데이터베이스 SQL 에디터를 통해 관리자 계정의 `role` 컬럼을 `'admin'`으로 직접 업데이트하여 안전하게 처리합니다.

## 3. 프론트엔드 UI (Frontend)
*   어드민 전용 레이아웃을 구성하여 일반 사용자 페이지와 시각적으로 명확히 분리합니다.
*   **대시보드 메인 화면**
    *   전체 유저 수 등 핵심 운영 통계 위젯을 최상단에 렌더링합니다.
*   **제보 대기열 및 승인 완료 테이블**
    *   대기 중(`pending`)인 제보 리스트와 이미 승인된(`active`) 합주실 목록을 탭 등의 방식으로 전환하여 제공합니다.
*   **검토 및 검증 모달**
    *   대기열 항목 클릭 시 상세 검토 팝업 모달이 뜹니다.
    *   모달 안에는 제보자 정보, 이름, 설명, 원본 네이버 지도 링크와 함께 **좌표 검증용 미니 네이버 지도**가 렌더링되어 파싱된 위치의 정확성을 육안으로 확인시켜 줍니다.
*   **액션 처리 및 반려 템플릿**
    *   [수락] 및 [거절] 버튼을 배치하여 상태를 직관적으로 변경할 수 있도록 UI를 구현합니다.
    *   거절 시에 열리는 모달에는 자주 쓰이는 반려 사유(예: "존재하지 않는 합주실입니다.", "네이버 지도 링크가 올바르지 않습니다.", "이미 등록된 동일 합주실이 존재합니다.")를 선택할 수 있는 상용구 클릭 버튼/드롭다운을 배치합니다. 이를 선택하면 텍스트 박스에 자동 입력되며 추가 편집도 가능한 하이브리드 입력 폼을 제공합니다.
    *   승인 완료된 테이블 행에는 즉각 서비스를 내릴 수 있는 [삭제] 버튼을 배치합니다.
